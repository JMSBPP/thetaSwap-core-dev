**Step 1 — Cargo.toml (1-line edit, must happen before any cli.rs code compiles):**

Change the clap line from:
```toml
clap = { version = "4.5", features = ["derive"] }
```
to:
```toml
clap = { version = "4.5", features = ["derive", "env"] }
```

Without the `"env"` feature, `#[arg(env = "ETH_RPC_URL")]` compiles but silently does NOT read env vars at runtime (confirmed from clap's Cargo.toml features table in research). This is the only Cargo.toml change needed.

**Step 2 — Create d2p/src/cli.rs:**

Write the full file following the exact structure from RESEARCH.md Pattern 1. Key points:

1. Only `Cli` derives `Parser`. Mid-level structs (`TsArgs`) derive `Args`. Subcommand enums derive `Subcommand`. `Protocol` derives `ValueEnum`. (Pitfall 2: never use `Parser` on mid-level structs.)

2. `#[command(name = "d2p", version, about = "ThetaSwap reactive contract deployment tool")]` on `Cli`. The `version` attribute reads `CARGO_PKG_VERSION` at compile time — do not hardcode a version string. (CMD-08)

3. `#[command(arg_required_else_help = true)]` on `Cli` — shows help when run bare with no subcommand.

4. `ReactiveArgs` must have `#[command(long_about = "...")]` containing a concrete example:
```
Deploy a reactive contract.

Example:
  d2p ts reactive uniswap-v3 \
    --rpc-url https://rpc.sepolia.org \
    --private-key $ETH_PRIVATE_KEY \
    --callback 0xcallback
```
(CMD-07)

5. `Protocol` enum:
```rust
#[derive(Clone, ValueEnum)]
pub enum Protocol {
    #[value(name = "uniswap-v3")]
    UniswapV3,
}
```
(Pitfall 3: use ValueEnum, not String — clap validates and shows allowed values in help)

6. `ReactiveArgs` fields:
- `pub protocol: Protocol` — positional, no `#[arg(...)]` needed
- `#[arg(long, env = "ETH_RPC_URL")] pub rpc_url: String` (CMD-02)
- `#[arg(long, env = "ETH_PRIVATE_KEY")] pub private_key: String` (CMD-03)
- `#[arg(long)] pub callback: String` — no default = required, clap enforces (CMD-04)
- `#[arg(long, default_value = "10react", value_parser = parse_value)] pub value: String` (CMD-05)
- `#[arg(long, default_value = ".")] pub project: PathBuf` — default is `"."` (string), not `PathBuf::from(".")` (Pitfall 4) (CMD-09)

7. `parse_value` free function (not a method, not in a struct):
```rust
fn parse_value(s: &str) -> Result<String, String> {
    let units = ["react", "ether", "gwei", "wei"];
    for unit in units {
        if let Some(num_str) = s.strip_suffix(unit) {
            if !num_str.is_empty() && num_str.parse::<f64>().is_ok() {
                return Ok(s.to_string());
            }
        }
    }
    Err(format!(
        "invalid value '{}': expected <number><unit>, unit one of: {}",
        s,
        units.join(", ")
    ))
}
```

8. No `--legacy` field anywhere in any struct. (CMD-06, `--legacy` is baked into primary.rs already.)

```rust
mod cli;
mod deploy;
mod errors;

use anyhow::Context;
use clap::Parser;
use cli::{Cli, Commands, Protocol, TsCommands};
use deploy::{DeployParams, Runner};

fn main() {
    if let Err(e) = run() {
        eprintln!("error: {e:#}");
        std::process::exit(1);
    }
}

fn run() -> anyhow::Result<()> {
    let cli = Cli::parse();
    match cli.command {
        Commands::Ts(ts) => match ts.command {
            TsCommands::Reactive(args) => {
                let contract_path = match args.protocol {
                    Protocol::UniswapV3 => "src/fee-concentration-index-v2/protocols/uniswap-v3/UniswapV3Reactive.sol:UniswapV3Reactive",
                };
                let project_dir = args.project.canonicalize()
                    .with_context(|| format!("project path does not exist: {}", args.project.display()))?;
                let params = DeployParams {
                    rpc_url: args.rpc_url,
                    private_key: args.private_key,
                    callback: args.callback,
                    value: args.value,
                    contract_path: contract_path.to_string(),
                    project_dir,
                };
                let output = Runner::new(params).deploy()?;
                println!("{output}");
                Ok(())
            }
        },
    }
}
```

Key implementation notes:

1. Use `fn main() + fn run() -> anyhow::Result<()>` split (RESEARCH Pattern 4). Do NOT use `fn main() -> anyhow::Result<()>` directly — the `run()` split gives `eprintln!("error: {e:#}")` control over stderr format (OUT-02) and explicit `std::process::exit(1)` for exit code (OUT-03).

2. `args.project.canonicalize()` fails with a clear message if path does not exist (Pattern 5, CMD-09). The `.with_context(|| ...)` adds the path to the error so the user sees which path was invalid.

3. The `contract_path` match must be exhaustive. `Protocol::UniswapV3` maps to exactly `"src/fee-concentration-index-v2/protocols/uniswap-v3/UniswapV3Reactive.sol:UniswapV3Reactive"`. This string is what `fallback::read_bytecode()` uses to find `out/UniswapV3Reactive.sol/UniswapV3Reactive.json`. Do not shorten it.

4. `println!("{output}")` — `DeployOutput` implements `Display` as `address\ntx_hash`. This satisfies OUT-01 and OUT-04 (no diagnostic noise on stdout).

5. No `use crate::errors::D2pError` needed — anyhow wraps D2pError via `?` automatically.

6. `mod cli;` must be added at the top — it did not exist in the stub.
