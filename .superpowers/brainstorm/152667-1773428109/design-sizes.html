How the logomark simplifies at each breakpoint

Simplification Strategy

As size decreases, detail is progressively removed to maintain clarity. Three tiers:

Hero / Large
64px+

Nav / Header
24–48px

Favicon
16px

Token Icon
32–48px circle

Tier Rules

Tier Size Elements Stroke Weight
Hero 64px+ All 4 elements (both arcs, lime bisector, ghost crossbar) 2.5 / 1.5 units
Nav 24–48px All 4 elements, thicker strokes for legibility 3.5 / 2 units
Favicon 16px 3 elements only — drop ghost crossbar 6 units
Token 32–48px 3 elements in circular container (stone-800 fill, stone-700 border) 4 units
